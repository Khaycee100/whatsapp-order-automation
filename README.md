# WhatsApp Business Order Tracker
A Python-based automation tool designed to bridge the gap between informal WhatsApp commerce and structured data management.

## 🚀 The Problem
Many small vendors in Lagos manage orders via WhatsApp, leading to lost messages and manual entry errors when moving data to Excel.

## 🛠️ The Solution (Work in Progress)
This script automates the capture of order details. 
- **Current Phase:** Local Python script using Pandas for CSV logging.
- **Next Phase:** Integrating **Twilio WhatsApp API** and **Flask webhooks** to capture live messages in real-time.

## Tech Stack
- Python 3.x
- Pandas (Data Structuring)
- CSV/Excel (Storage)
